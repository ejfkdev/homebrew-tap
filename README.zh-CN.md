# ejfkdev/homebrew-tap

[English](README.md)

ejfkdev CLI 工具的 Homebrew tap 仓库。

## 使用方法

```bash
# 直接安装
brew install ejfkdev/tap/dj
brew install ejfkdev/tap/udf
brew install ejfkdev/tap/apix
brew install ejfkdev/tap/tspc

# 或者先添加 tap，再安装
brew tap ejfkdev/tap
brew install dj
```

## 可用软件

| 软件 | 版本 | 说明 |
|------|------|------|
| [dj](https://github.com/ejfkdev/dj) | 0.2.0 | CLI 工具 |
| [udf](https://github.com/ejfkdev/udf) | 0.1.0 | CLI 工具 |
